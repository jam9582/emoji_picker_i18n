## 2026-07-09 ② 데이터 파이프라인 (`tool/generate_data.dart`)

- **소스**: emojibase-data 16.0.3 compact.json (jsDelivr CDN에서 다운로드)
- **필터**: group이 없거나(국기 문자 조각 등 26개) group 2(피부색 부품 9개)인 항목 제외 → **피커 대상 1,906개**. 전부 order·tags 보유 확인
- **출력 형식** (기획서 5-3절 "공통 1벌 + 언어별 키워드" 구현):
  - `lib/src/data/emoji_common.dart`: `'unicode|group'` 또는 `'unicode|group|skin1,skin2,...'` — 표시 순서(order)로 정렬, 59KB
  - `lib/src/data/emoji_locale_<xx>.dart`: `'label|tag1|tag2|...'` — 공통과 같은 인덱스, ko 127KB / en 103KB
  - `lib/locales/<xx>.dart`: 사용자용 공개 import 경로 (내부 파일 재export)
- **인덱스 정합성**: 공통은 en 기준으로 만들고 각 언어는 hexcode로 매칭 (언어 간 순서 차이 방어)
- **파이프라인 안에서 배운 것**: 구분자 `|`가 데이터에 있으면 즉시 예외 — 침묵 손상 방지
- 다운로드 캐시(`tool/cache/`)는 재생성 가능하므로 gitignore

### 여기서 발견한 함정: 이모지 표시 지정자(U+FE0F)

- 테스트에서 `contains('🐈')`가 실패 → 데이터의 🐈는 투명한 "컬러로 그려라" 지시문자(U+FE0F)가 붙은 2글자였음
- 원인·해결법 후보·적용 범위는 기획서 5-1절에 상세 기록. 결론: 데이터는 원본 유지, 출처가 다른 문자열이 만나는 지점(선택값 강조·역조회·최근사용 중복 제거)에서만 정규화

## 2026-07-09 ③ 한글 초성·자모 유틸 (`lib/src/search/hangul.dart`)

- 한글 음절은 `0xAC00 + 초성×588 + 중성×28 + 종성` 공식으로 분해 → **사전 데이터 0바이트**
- 두 가지 표현을 생성:
  - 초성 문자열: '고양이' → 'ㄱㅇㅇ' (초성만 친 검색어용)
  - **자모 스트림**: '고양' → 'ㄱㅗㅇㅑㅇ' (그 외 모든 한글 검색어용)
- 매칭 규칙: 검색어가 초성으로만 구성 → 초성 문자열 포함 검사 / 그 외 → 양쪽을 자모 스트림으로 펴서 부분 문자열 검사
- **자모 스트림 방식의 보너스**: '고ㅇ'(ㄱㅗㅇ)과 타이핑 중간 상태 '공'(ㄱㅗㅇ)이 같은 스트림이 되어, 한글 입력기의 중간 조합 상태에서도 자연스럽게 검색됨. 별도 처리 없이 공짜로 얻은 기능
- 겹받침(값→ㄱㅏㅂㅅ)·겹모음(화→ㅎㅗㅏ)은 키보드 타이핑 순서로 분해 — 양쪽을 같은 규칙으로 펴기 때문에 정합성 유지

## 2026-07-09 ④ 검색 엔진 (`lib/src/search/emoji_search.dart`)

- 구조: 생성 시 색인 1회 구축(키워드 소문자화 + 한글이면 초성·자모 스트림 캐시) → 검색은 선형 스캔이지만 1,906×키워드 수 규모라 1ms 미만
- 여러 언어를 넘기면 **모든 언어 동시 검색** (kEmojiLocaleKo + kEmojiLocaleEn → '고양이'도 'cat'도 매칭)
- `findByChar()`: 정규화된 문자열 → Emoji 역조회 맵. 기획서 5-1절 적용 지점 ⓑ(선택값 강조)·ⓒ(피부색 역조회) 담당
- `normalizeEmoji()` / `sameEmoji()`: FE0F/FE0E 제거 비교 유틸을 공개 API로 노출

### 순위 체계 결정 (기획서 8절 미해결 항목 하나 해소)

- 첫 구현에서 '고양이' 검색의 1등이 🐈(이름=고양이)가 아니라 😺(태그에 고양이)로 나오는 문제 발견
  - 원인: 이름 정확 일치와 태그 정확 일치가 동점 → 표준 순서상 앞인 😺가 승리
- 결정한 순위: **정확 > 접두 > 포함의 3단계 × 각 단계에서 이름 > 태그** = 6단계
  (짝수=이름, 홀수=태그 점수로 구현. 동점이면 유니코드 표준 순서)
- 결과: '고양이' → 🐈(정확·이름) → 😺😸...(정확·태그) → '고양이 얼굴'(접두·이름) → '검은 고양이'(포함·이름) 순
